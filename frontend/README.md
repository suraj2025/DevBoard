# DevBoard
A modern React + TypeScript task management application built to master React ecosystem tools including MSW, React Query, Zustand, React Hook Form, and Zod.
DevBoard
DevBoard is a modern task and habit tracking application built with React and TypeScript. The project is designed as a hands-on learning journey to master professional frontend development practices and tools used in production applications.

Features
Authentication system

Task management (CRUD)

Habit tracking

Form validation with Zod

Global state management with Zustand

API data fetching and caching with React Query

Mock backend using MSW (Mock Service Worker)

Protected routes

Type-safe development with TypeScript

Responsive modern UI

Testing-ready architecture

Tech Stack
React

TypeScript

React Router

Zustand

React Query (TanStack Query)

React Hook Form

Zod

MSW (Mock Service Worker)

Tailwind CSS

Vitest

Testing Library

This project focuses on:

Production-level frontend development
