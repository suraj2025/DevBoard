# DevBoard

A full-stack personal productivity app for tracking tasks and habits, built as a learning project covering modern React patterns, Spring Boot REST APIs, JWT auth, and Docker deployment.

## Tech Stack

**Frontend**
- React 18 + TypeScript
- Vite
- TanStack Query (React Query v5)
- Zustand (global state)
- React Hook Form + Zod (form validation)
- Tailwind CSS v4
- Recharts (analytics charts)
- Axios

**Backend**
- Spring Boot 3
- Spring Security + JWT
- PostgreSQL
- Hibernate / JPA
- Maven

**DevOps**
- Docker + Docker Compose
- Vercel (frontend)
- Railway (backend)

## Features

- JWT authentication (login, register, logout)
- Kanban task board (todo, in progress, done)
- Daily habit tracker with streaks and 7-day grid
- Reports & analytics with charts
- Settings (profile, notifications, account security)
- Protected routes
- Persistent auth (Zustand + localStorage)
